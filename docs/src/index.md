# Gridworld.jl

Welcome to the documentation for Gridworld.jl!

This package provides a simple implementation of a Gridworld environment for reinforcement learning experiments. Modeled after FrozenLake environment from OpenAI.

## Quick Links

- [Installation](installation.md)
- [Usage](usage.md)

## Features

- Customizable grid size
- Configurable start and goal positions
- Configurable optional "holes" in the grid
- Deterministic and stochastic options
- Manual movement test mode

For more details, please check out the [Usage](usage.md) section.
